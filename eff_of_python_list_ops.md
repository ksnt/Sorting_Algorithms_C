# Efficiency of Operations

## Big-O Efficiency of Python List Operators

| Operation   | Big-O Efficiency |
|:------------|:-------------|
| indexx[]    | O(1)         |
| append      | O(1)         |
| pop()       | O(1)         |
| pop(i)      | O(1)         |
| insert(i,item) | O(n)      | 
| del operator   | O(n)      |
| iteration      | O(n)      | 
| contains(in)   | O(n)      |
| get slice [x:y]| O(k)      |
| del slice      | O(n)      |
| set slice      | O(n+k)    |
| reverse        | O(n)      |
| concatenate    | O(k)      |
| sort           | O(nlogn   |
| multiply       | O(nk)     |

## Big-O Efficiency of Python Dictionary Operations

| Operation   | Big-O Efficiency |
|:------------|:-------------|
| copy        | O(n)         |
| get item    | O(1)         |
| set item    | O(1)         |
| delete item | O(1)         |
| contains(in)| O(1)         |
| iteration   | O(n)         |
